Detailed Description

Represents a 2D arc segment defined by a circular arc between two points.

Stores geometric data such as center, radius, orientation, and start/end angles. Can be created from endpoints or directly via polar form. Supports movement, recalculation, and conversion from polar to Cartesian coordinates.
