Contour2D polylineContourFromPoints ( std::vector< MyPoint > & ptr,
bool closedContour )

Constructs a Contour2D from a list of points interpreted as a polyline.

Creates a contour by connecting each consecutive pair of points with a LineSegment2D. Optionally closes the contour by connecting the last point back to the first.

Parameters
ptrVector of points to connect.
closedContourIf true, adds a segment from the last point to the first.
Returns
A Contour2D composed of straight segments between the given points.
